## 🌸 What is HerCalm?

HerCalm is a privacy-first Progressive Web App that helps women manage period pain, PMS, and stress using gentle, science-backed tools. No account, no tracking. Data stays on your device.

**Live Demo:** [hercalm.app](https://wytzbot.github.io/hercalm)

## ✨ Features

### **8 Core Tools**
1. **🧘‍♀️ Yoga Flow** - 6-minute guided poses for pelvic relief
2. **🌸 Calm Breath** - 4-7-8 breathing to reduce cortisol
3. **🔥 Heat Timer** - Safe 20-min heat therapy timer
4. **💆‍♀️ Pressure Points** - SP6 + LI4 acupressure with timers
5. **📊 Tracker** - Log pain, symptoms, notes. See patterns
6. **😊 Mood Check** - Daily PMS/mental health check-in
7. **⏱️ Mindful Timer** - Rain, ocean, white noise breaks
8. **🍵 Tea Guide** - Ginger, chamomile, peppermint for cramps

## 🔐 Privacy First

| **Data stored locally only** | Data sent to cloud/servers |
| **No account needed** | Email/login required |
| **No analytics/trackers** | Tracks cycle for advertisers |

## 🌐 Online Required

HerCalm requires an internet connection to work. This allows:
1. **Free access** - Supported by ads, no subscription needed
2. **Always updated** - Latest tools and content
3. **Fast loading** - < 200KB total size

If you're offline, you'll see a friendly message to reconnect.
